Component: tos.chips.at45db.HplAt45dbByteC

generic module HplAt45dbByteC(int sectorSizeLog2)

Generic byte-at-a-time implementation of the AT45DB HPL. Each platform must provide its own HPL implementation for its AT45DB flash chip. To simplify this task, this component can easily be used to build an AT45DB HPL by connecting it to a byte-at-a-time SPI interface, and an HplAt45dbByte interface.

Parameters:
The - number of bits needed to represent a sector size, e.g., 9 for the AT45DB041B.
